# Angry Button - Humor for When You're Mad

This project is a simple, lightweight web app designed to help users channel their anger with a bit of humor. By clicking the big, bold “I AM ANGREYYYY” button, users receive a random funny or lighthearted message to lighten the mood.

The app emphasizes ease of use, quick feedback, and fun interaction without any distractions or complex setup.

---

## Key Features

1. Single-page interface with a prominently displayed angry button.  
2. On each click, displays a random humorous message to the user.  
3. Clean, centered layout with responsive styling for a smooth user experience.  

---


## Data Sources

- [JokeAPI](https://v2.jokeapi.dev/) – free public API providing a variety of jokes used for dynamic message fetching.
